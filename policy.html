Kebijakan pemrosesan data pribadi 1. Ketentuan Umum Kebijakan pemrosesan data pribadi ini dikompilasi sesuai dengan persyaratan hukum Federal Hukum No. 152-FZ dari 27.07.2006" pada data pribadi " dan mendefinisikan prosedur untuk memproses data pribadi dan mengukur untuk memastikan keamanan data pribadi data Mikhalov Ivan Sergeevich (hereinafter-the Operator). Operator bertujuan dan realisasi dari kegiatan-nya hak dan kebebasan manusia dan warga selama pengolahan data pribadi, termasuk perlindungan hak kekerasan kehidupan pribadi, pribadi dan privasi keluarga. Real kebijakan Operator pada pemrosesan data pribadi (sininafter disebut sebagai kebijakan) berlaku untuk semua informasi, Operator yang bisa menerima pengunjung situs. .

Konsep dasar yang digunakan dalam Kebijakan Proses otomatis dari data pribadi pemrosesan data pribadi menggunakan teknologi komputer; Memblokir data pribadi-sementara waktu pemrosesan data pribadi (kecuali untuk kasus ketika pemrosesan diperlukan untuk mengklarifikasi data pribadi); Situs web-satu set Bahan grafis dan informasi, serta program komputer dan basis data yang menyediakan ketersediaan mereka di Internet di alamat jaringan ;

Sistem Informasi Data Pribadi - satu set data pribadi yang terkandung dalam basis data, dan menyediakan pengolahan informasi teknologi dan sarana teknis mereka; Penghapusan data aksi pribadi yang tidak dapat ditentukan tanpa penggunaan tambahan informasi milik data pribadi ke pengguna tertentu atau subjek data pribadi lainnya; Pemrosesan data pribadi – aksi apapun (operasi) atau set aksi (operasi) dilakukan dengan atau tanpa perkakas otomatisasi data pribadi, termasuk koleksi, rekaman, systematization, akumulasi, storage, perbaikan( update, change), extraction, use, transfer (distribusi, penyediaan, akses), penghapusan, penghancuran data pribadi; Operator ... otoritas kota ... , hukum entitas atau individu, independen atau jointly dengan orang lain yang mengatur dan memproses data pribadi, serta menentukan tujuan pengolahan data pribadi, komposisi data pribadi untuk diproses, aksi (operasi) dilakukan dengan data pribadi data; Data pribadi – informasi terkait secara langsung atau tidak langsung ke Pengguna spesifik atau dapat diidentifikasi situs web ;

Pengguna - pengguna situs web ;

Ketentuan data pribadi yang ditujukan untuk mengungkap data pribadi seseorang. untuk seseorang atau lingkaran orang tertentu; Penyebaran data pribadi-setiap tindakan yang ditujukan untuk pengungkapan data pribadi kepada orang yang tak dikenal untuk masyarakat umum (transfer data pribadi) atau untuk berkenalan dengan data pribadi dari sejumlah orang yang tak terbatas, termasuk pengungkapan data pribadi di media massa, penempatan dalam informasi dan jaringan telekomunikasi atau menyediakan akses ke data pribadi dengan cara lain; Transfer data-lintas perbatasan pribadi data pribadi pada wilayah negara asing ke otoritas otoritas negara asing, individu asing atau badan hukum asing; Penghancuran data pribadi-setiap tindakan yang menghasilkan data pribadi yang tidak dapat ditarik kembali dihancurkan dengan kemustahilan lebih lanjut mengembalikan isi data pribadi dalam Sistem Informasi Data Pribadi dan sebagai akibat dari hal itu, semua data pribadi akan hancur. Operator dapat memproses data pribadi Pengguna berikut Nama keluarga, nama, patronymic; Alamat surel; Nomor telepon; Tahun, bulan, tanggal dan tempat lahir; Foto; Juga di situs web, koleksi dan pengolahan data anonim tentang pengunjung (termasuk "Kuki") melalui Layanan Statistik Internet (metrik Yandex dan Google Analytics dan lain-lain). Data yang disebutkan di atas lebih banyak dikelompokkan di bawah konsep umum Data Pribadi. Tujuan memproses data pribadi Tujuan pengolahan data pribadi pengguna adalah untuk menginformasikan pengguna dengan mengirim email. menyediakan pengguna dengan akses ke Layanan, Informasi dan / atau bahan yang terkandung di situs. Operator juga memiliki hak untuk mengirim pemberitahuan ke pengguna tentang produk baru dan layanan, penawaran khusus dan berbagai peristiwa. Pengguna dapat selalu menolak untuk menerima pesan informasi dengan mengirim surel ke Operator pada alamat email Kebijakan@

Data pengguna terpisah yang dikumpulkan melalui Layanan Statistik Internet digunakan untuk mengumpulkan informasi tentang aksi pengguna di situs tersebut, meningkatkan kualitas situs dan isinya. Alasan hukum untuk memproses data pribadi Operator memproses data pribadi Pengguna hanya jika mereka diisi dan / atau dikirim oleh pengguna secara independen melalui bentuk-bentuk khusus terletak di situs . Dengan mengisi formulir dan / atau mengirim data pribadi Anda ke Operator, Pengguna setuju dengan kebijakan ini. Operator proses depersonalized data tentang Pengguna bila hal ini diijinkan dalam pengaturan peramban pengguna (menyimpan kuki dan menggunakan kuki diaktifkan JavaScript technologies). Prosedur untuk mengumpulkan, menyimpan, mentransfer dan proses data pribadi lainnya Keamanan data pribadi yang diproses oleh Operator adalah dijamin dengan menerapkan hukum, penyelenggara dan langkah-langkah teknis yang diperlukan untuk sepenuhnya mematuhi persyaratan dari undang-undang saat ini undang-undang di bidang perlindungan data pribadi. Operator memastikan keselamatan data pribadi dan mengambil semua langkah-langkah yang memungkinkan untuk mencegah akses ke data pribadi orang yang tidak sah. Data pribadi Pengguna tidak akan pernah, dalam keadaan apapun, dipindahkan ke pihak ketiga, dengan pengecualian kasus yang berhubungan dengan implementasi undang-undang saat ini. Jika data pribadi tidak akurat terdeteksi, pengguna dapat mengupdate dengan mengirimkan notifikasi ke kebijakan alamat surel Operator@ ditandai"memperbarui data pribadi". Istilah untuk memproses data pribadi tidak terbatas. Pengguna dapat menarik persetujuan mereka untuk pengolahan data pribadi setiap saat dengan mengirimkan pemberitahuan ke Operator melalui e-mail ke kebijakan alamat Operator@ ditandai"pembatalan persetujuan untuk pengolahan data pribadi". Pemindahan data pribadi lewat perbatasan. Operator berkewajiban untuk memastikan bahwa negara asing untuk mentransfer data pribadi yang seharusnya terjadi adalah negara asing sebelum transfer lintas-batas data pribadi, perlindungan yang dapat diandalkan dari hak-hak Data Pribadi subyek disediakan. Pemindahan data pribadi di wilayah negara-negara asing tidak memenuhi syarat tersebut hanya dapat dilakukan jika data pribadi subjek ditulis untuk transfer lintas-batas data pribadi dan / atau kinerja kontrak yang Subjek Data Pribadi adalah pesta. Ketentuan akhir Pengguna dapat memperoleh klarifikasi apapun pada isu-isu yang berkaitan dengan pemrosesan data pribadi mereka dengan menghubungi Operator melalui kebijakan e - mail@. Dokumen ini akan mencerminkan perubahan apapun untuk kebijakan pemrosesan data pribadi Operator. Kebijakan valid tanpa batas waktu sampai digantikan dengan versi baru.
